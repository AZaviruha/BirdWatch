What is this page?

You are looking at Birdwatch, an open source side project for tweet stream analysis. The application subscribes to all tweets that contain a certain term, e.g. Ferguson. All matching tweets are available for further search and analysis. Every search is updated in (near) real time. Typically, when you tweet something containing the term, you should see your tweet appear in the latest section within one or two seconds.

Multiple versions of this application exist, each of which is written in different languages and technologies. The main idea behind this application is to play around with different ways of programmatically reasoning about a stream of information.

Technology Stack

The incarnation of the application that you are looking at right now has been written in Clojure on the server side as well as ClojureScript and Om, plus D3, plain ReactJS and Rickshaw for the graphs / charts.

Usage

When the application starts, it loads the last 2000 tweets matching a search, with * for matching all tweets as the default search. The search result will then be updated in real time. In the navigation bar, you can see the total count of tweets currently loaded. On the left side, tweets are displayed in different sort orders:

  • latest: by date in descending order (newest on top)
  • followers: by the number of followers of the tweet's author
  • retweets: by the total retweets of a tweet
  • retweets2: by the number of retweets within the loaded tweets
  • favorites: by total number of times a tweet has been favorited

You can respond to, retweet and favorite a tweet directly from this application by using the little icons on the lower right of the tweet frames.

On the right side there are several charts. On top is a time series chart that displays how the search results were distributed over time. Below is a word cloud with the size of a word signifying how often it appeared in the results. Below is a bar chart showing the top 25 words and their individual trend over time. The words in both the word cloud and the bar chart are clickable, with the word then appearing in the search bar. The search bar initiates a new, refined search. At the same time, the **URL** changes so that you can bookmark a specific search or share a link.
