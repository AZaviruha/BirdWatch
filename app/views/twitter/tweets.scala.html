BirdWatch

This is a sample reactive web application making use of Play Framework 2.1 visualizing a live stream of tweets from the Twitter Streaming API. It continuously retrieves all Tweets that contain the following words from Twitter:

    @topics.map { topic =>
  • @topic
    • }

You can click on the words either in the word cloud or in the bar chart to drill into the data. Clicking on a word again that is already in the current query will replace the query to just look for that word. You can see the current query in the address field in your browser and you can also bookmark that address. It's a resource.

Check out my Blog or the GitHub project for more information on how this is implemented.

You need a modern browser to view this page, otherwise the content will not be dynamic.
